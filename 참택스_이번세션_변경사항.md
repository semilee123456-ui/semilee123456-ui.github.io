## 2. 레이아웃 정리 (디자인 브리프 + 사용자가 전달한 "새 레이아웃" 참고)

사용자가 다른 도구(추정 Claude Design 계열)에서 받은 레이아웃 시안(`참택스_공유용.html`, 실제로는 폰트 등을 base64로 번들링한 스냅샷 파일이었음 — 안에 `<script type="__bundler/template">`로 실제 페이지 구조가 JSON 인코딩되어 있어서 그걸 추출해서 대조함)을 실제 코드와 항목별로 비교했습니다.

**주의**: 이 시안은 확인해보니 지금 실제 코드보다 오히려 이전 단계의 초안이었고, FAQ 검색·카테고리 필터·숨은 돈 찾기 체크리스트·환급 위저드·문의 폼(Formspree)·잭팟 상세 드로어·다국가 확장 구조(`COUNTRY_TAX_PROFILES`)·마일스톤 효과 등 **핵심 기능이 다수 빠져 있었음**. 시안의 "문의하기"는 개인 네이버 이메일이 `mailto:` 링크로 그대로 노출되는 방식이라, 절대 이대로 반영하면 안 됨 — 실제 코드의 Formspree 폼을 유지해야 함.

따라서 시안에서는 **시각적 구조·문구·아이콘 선택 등 표면적인 부분만** 골라서 반영했고, 기능은 전부 실제 코드 그대로 유지했습니다.
